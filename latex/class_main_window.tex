\doxysubsection{Detailed Description}
Główne okno aplikacji monitorującej jakość powietrza. 

Klasa ta zarządza interfejsem użytkownika, wyświetla dane pomiarowe, wykresy i analizy, oraz obsługuje interakcje z użytkownikiem.
